# 📁 File Integrity Monitor

A web-based cybersecurity project built using Flask and Python that monitors files and detects integrity changes using SHA-256 hashing. The system creates a baseline of trusted files and identifies modifications, additions, and deletions through an interactive dashboard.

---

## 🚀 Features

✅ Create file baseline

✅ Monitor files and directories

✅ Detect modified files

✅ Detect newly added files

✅ Detect deleted files

✅ SHA-256 hash-based verification

✅ Interactive cyber-themed dashboard

✅ Real-time integrity checking

---

## 🛠 Tech Stack

* Python
* Flask
* SHA-256 (hashlib)
* JSON
* HTML
* CSS
